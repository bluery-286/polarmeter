PolarMeter 개인정보처리방침 초안

작성일: 2026-06-12

상태: V1-04 RC / 공개 전 현진이형 승인 필요

이 문서는 PolarMeter TestFlight/App Store 제출 준비용 초안입니다. 실제 공개 URL 게시 전 최종 승인과 연락처 확인이 필요합니다.

1. 서비스 개요

PolarMeter는 미국장과 국내장의 시장 환경을 온도처럼 요약해 보여주는 시장 브리핑 앱입니다.

PolarMeter는 투자 권유, 거래 지시, 개인 맞춤형 금융 자문, 수익 보장을 제공하지 않습니다.

2. 수집하는 개인정보

V1-04 RC 기준 PolarMeter 앱은 사용자의 개인정보를 직접 수집하지 않습니다.

현재 앱이 직접 수집하지 않는 정보는 다음과 같습니다.

  • 이름, 이메일, 전화번호 등 식별 정보
  • 증권 계좌, 포트폴리오, 거래 내역
  • 결제 정보
  • 위치 정보
  • 연락처, 사진, 마이크, 카메라 데이터
  • 광고 식별자 또는 맞춤형 광고용 정보

향후 로그인, 알림, 관심종목, 유료 기능, 문의 기능 등을 도입하는 경우 수집 항목과 이용 목적을 별도로 고지하고 필요한 동의를 받겠습니다.

3. 자동으로 처리될 수 있는 정보

V1-04 RC 기준 PolarMeter는 다음 SDK를 포함하지 않습니다.

  • 광고 SDK 없음
  • 결제 SDK 없음
  • 증권사/브로커리지 계정 연결 없음
  • 외부 분석 SDK 없음

다만 앱 배포와 테스트 과정에서 Apple TestFlight, App Store, 운영체제, 네트워크 제공자가 기본 진단 정보나 사용 정보를 처리할 수 있습니다. 이 정보는 각 제공자의 개인정보 처리방침을 따릅니다.

PolarMeter는 자체 서버로 사용자를 식별할 수 있는 정보를 전송하지 않습니다.

4. 이용 목적

현재 PolarMeter가 직접 수집하는 개인정보가 없으므로 별도 개인정보 이용 목적도 없습니다.

향후 기능 확장으로 개인정보를 처리하게 되는 경우 다음 목적 범위 안에서 최소한으로 처리합니다.

  • 서비스 제공 및 오류 대응
  • 사용자 설정 저장
  • 알림 제공
  • 고객 문의 대응
  • 법령 또는 스토어 정책 준수

5. 제3자 제공 및 처리위탁

V1-04 RC 기준 PolarMeter는 개인정보를 제3자에게 제공하지 않으며, 개인정보 처리 업무를 외부 업체에 위탁하지 않습니다.

앱 배포와 베타 테스트 과정에서 Apple TestFlight/App Store가 처리하는 정보는 Apple의 정책을 따릅니다.

6. 데이터와 뉴스에 관한 고지

PolarMeter는 시장 환경 이해를 돕기 위해 preview/stub 데이터, 지연 데이터, 무료 공개 데이터 실험, 뉴스 제목/출처/발행시각 등 제한된 정보를 표시할 수 있습니다.

  • 뉴스 원문 전문을 저장하거나 미러링하지 않습니다.
  • 증권 계좌나 개인 포트폴리오를 연결하지 않습니다.
  • 데이터는 지연되거나 제공자 사정에 따라 일시적으로 확인되지 않을 수 있습니다.
